----

Description

Cet outil permet de voir les relations entres les personnes

Il permet de regrouper les informations dans une base unique

Utilisation

Fonctionnant à travers un serveur web un simple browser permet d'effectuer la plupart des opérations

View

permet de voir les contacts existants

Add

permet d'ajouter des contacts ou des attributs

Search

Permet la recherche d'un personne par son nom, son prenom ou simplement avec une phrase, un mot, ou bien un lettre

Possibilités

  • Exporter la base de données dans un simple fichier

  • Utiliser plusieurs base de données dans le cadre d'une utilisation avec plusieurs utilisateurs

  • De faire un calendrier de tous les anniversaires

  • Gerer completement les rendez vous...
